# Project Vue.js

This is my personal project to learn and practice, using the vue.js framework, based on studies in [Udemy course] author "[David Katz]".

### Install vue-cli

```sh
 $ npm install vue-cli -g

```

### Start a new project

```sh
$ vue init  webpack-simple
$ cd startbase
$ npm install
$ npm run dev
```
